TikTok ร้านอาหาร 46 สไลด์ HTML (เล่นคลิปได้)
============================================

ไฟล์หลัก: restaurant-tiktok-course.html

วิธีเปิด:
1. unzip ไฟล์นี้ลงโฟลเดอร์ใดก็ได้
2. ดับเบิ้ลคลิก restaurant-tiktok-course.html
3. ใช้ Browser ที่ทันสมัย (Chrome, Edge, Safari)

คีย์ลัด:
- ←  / →    : เลื่อนสไลด์ ก่อนหน้า / ถัดไป
- Space    : ถัดไป
- F        : เต็มหน้าจอ (Fullscreen)
- P        : เปิด/ปิด เลเซอร์พอยน์เตอร์
- Home/End : ไปสไลด์แรก / สุดท้าย
